Margens CSS

Como margin as propriedades CSS são usadas para separar o espaço em torno dos elementos, os fóruns de qualquer borda definida.

Com CSS, rápido Você tem Controle Sobre total em Margens. Existem propriedades para definir uma margem para cada lado de um elemento (superior, direita, inferior e esquerda).
